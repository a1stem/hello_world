# 👋 hello_world.py

> *Because every great journey starts with two words.*

---

## 🎉 What is this?

It's a **Hello World** — but make it fashion.

No boring `print("Hello, World!")` here. This is a fully animated, gold-tinted, geometric ASCII art rendition of the two most important words in any programmer's career.

Think of it as a **"first commit"** that actually deserves to exist.

---

## ✨ What makes it special?

- 🟡 **Gold & white two-tone ASCII art** — because plain text is so last century
- ⌨️ **Typewriter streaming effect** — each character appears one at a time, dramatically
- 🖤 **Minimal & elegant framing** — ruled lines, a quiet byline, generous breathing room
- 🐍 **Pure Python** — no dependencies, no fuss, just vibes

---

## 🚀 How to run it

```bash
python3 hello_world.py
```

That's it. Seriously.
